# 文档目录

## 截图说明

1. **demo-screenshot.png** - 主要功能演示截图
   - 展示拖拽网格组件的完整功能
   - 包含多种卡片类型和交互效果
   - 建议尺寸：1200x800 或更大

## 如何获取截图

1. 运行开发服务器：`npm run dev`
2. 访问：`http://localhost:5173`
3. 调整浏览器窗口到合适尺寸
4. 使用截图工具（如系统自带、浏览器开发者工具等）
5. 保存为 `demo-screenshot.png` 到 `docs/` 目录
6. 确保截图清晰展示以下功能：
   - 顶部工具栏和统计信息
   - 拖拽网格布局
   - 多种卡片类型（图表、表格、指标、图片）
   - 底部使用说明

## 截图要求

- **格式**: PNG（推荐）或 JPG
- **尺寸**: 最小 1200x800，推荐 1920x1080
- **质量**: 清晰、无模糊
- **内容**: 完整展示组件功能
